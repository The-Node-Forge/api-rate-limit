### Prerequisites

- `Node.js --version 18+`
- `npm or yarn`

Install the package using **npm** or **yarn**:

```sh
npm install @the-node-forge/api-rate-limit
```

or

```sh
yarn add @the-node-forge/api-rate-limit
```
